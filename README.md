# Welcome to the TempestAPI

This is a Node.js API for sample project

## Prerequisites

- Node.js
- TypeScript
- MongoDB
- Mongoose
- Typegoose
- Docker
- Redis
- Zod

## Getting Started

1. Clone the repository:
```sh 
git clone https://github.com/[username]/tempest.git
```

2. Install the dependencies:
```sh 
npm install
```

3. Start the development server:
```sh 
npm start
```
